Esta página web está dedicada al servidor de Minecraft de AoiiVT. Aquí podrás encontrar toda la información de como unirte al servidor, obtener datos sobre el número de jugadores que están conectados, el ping, el MOTD...

Este servidor fue creado para toda la comunidad de AoiiVT para que pudiesen estar con ella jugando, para que se formen nuevas amistades pero sobre todo, para que podais pasar tiempo con vuestra VTuber Española favorita!
